SharePost API

The SharePost API allows developers to fetch information of user profiles. The data that is visible in the API is never private or valuable information. It contains the username, bio, status, profile picture, and the number of posts & comments only.

How to use the API

To request information from the API, send a request to this URL: https://sharepost.jonathan2018.repl.co/api?profile=[@profile], where you replace the [@profile] with the profile name your are trying to retreive (including the @ symbol). For example, if I were to want to retreive the information from the account "@Sample", I would send a request to https://sharepost.jonathan2018.repl.co/api?profile=@Sample.

NOTE: The profile field is not case-sensitive.
